Tested on Windows 10, Python 2.7.13, Django 1.11.2
1. Install Redis Server: https://freefr.dl.sourceforge.net/project/redis/redis-2.6.10/redis-2.6.10-x86.zip
2. Start Redis ( C:\Redis\bin\redis-server.exe )
3. Run in new window: python manage.py run_huey
4. Run: python manage.py runserver 
4. Open URL: http://localhost:8000
5. Enjoy!
